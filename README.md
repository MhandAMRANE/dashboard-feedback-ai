# Dashboard Feedback AI

A dashboard for analyzing and visualizing user feedback using LLMs.

## Features
- Feedback upload and storage
- LLM-powered sentiment analysis and categorization
- Interactive dashboard for insights

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Configure environment: Copy `.env.example` to `.env` and add your API keys.
3. Run the app: `python app.py`
